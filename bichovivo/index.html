Rastreio de Olhos

O bichinho segue o movimento do seu cursor em tempo real, criando uma conexão visual única.

Sobre o projeto

BichoVivo é um projeto open-source que combina tecnologias web modernas para criar uma experiência nostálgica e interativa. Inspirado nos Tamagotchis dos anos 90, mas com recursos que só a web moderna pode oferecer.
