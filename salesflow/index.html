Gestao de Clientes

Gestao de Fornecedores

Controle de fornecedores com razao social, CNPJ validado, contato e categoria. Mascaras automaticas de CNPJ e telefone.

Pipeline de Vendas

Acompanhe cada etapa do ciclo comercial, do primeiro contato ao fechamento. Valores em BRL com formatacao automatica.

Assistente comercial inteligente powered by Google Gemini. Sugere acoes, analisa dados e oferece insights para decisoes estrategicas.

Autenticacao Segura

Login via Supabase com autenticacao segura. Cada usuario ve apenas seus proprios dados.

Tecnologia

Stack moderna e escalavel.

React 18
Supabase (Auth + Database)
Vite
Google Gemini AI
GitHub + Vercel

O SalesFlow foi construido com uma stack moderna que prioriza performance, escalabilidade e experiencia do desenvolvedor. O React 18 garante uma interface responsiva e componentizada, enquanto o Supabase oferece autenticacao robusta e banco de dados em tempo real sem a complexidade de um backend tradicional.

O Vite proporciona builds ultrarapidos e hot module replacement instantaneo durante o desenvolvimento. A integracao com o Google Gemini potencializa a assistente NOVA, que analisa dados comerciais e sugere acoes estrategicas com inteligencia artificial.

O deploy e feito automaticamente via GitHub para o Vercel, garantindo CI/CD continuo e disponibilidade global com edge computing.
